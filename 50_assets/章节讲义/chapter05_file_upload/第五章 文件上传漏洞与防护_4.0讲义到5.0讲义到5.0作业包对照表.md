## 4. 5.0 新增升级点

与 4.0 相比，第五章 5.0 讲义新增了这些系统约束：

1. `文件上传链路图/风险控制关系图` 被显式提升为收口证据，而不是项目附图。
2. `证据 -> 结论` 约束被前置到讲义层，避免学生把“上传成功”直接写成大结论。
3. `AI输出审核与人工复核记录` 被接入作业与讲义系统。
4. 在无正式班级反馈前，已预先接入课程级高频误区：
   - 关系链不足
   - 证据不清晰
   - 证据与结论不对应
