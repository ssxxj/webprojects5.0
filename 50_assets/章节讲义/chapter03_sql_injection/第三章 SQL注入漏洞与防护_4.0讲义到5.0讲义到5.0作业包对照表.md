## 4. 5.0 新增升级点

与 4.0 相比，第三章 5.0 讲义新增了这些系统约束：

1. `SQL注入因果链/防护关系图` 被显式提升为收口证据，而不是项目附图。
2. `证据 -> 结论` 约束被前置到讲义层，避免学生把现象直接写成根因。
3. `AI输出审核与人工复核记录` 被接入作业与讲义系统。
4. 当前已预先接入课程级高频误区：
   - 任务部分完成
   - 证据不清晰
   - 证据与结论不对应
   - 关系链不足
   - 边界表达不足
