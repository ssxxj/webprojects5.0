## 4. 5.0 新增升级点

与 4.0 相比，第八章 5.0 讲义新增了这些系统约束：

1. `统一分析框架图/项目论证结构图` 被显式提升为收口证据，而不是项目附件。
2. `证据 -> 结论` 约束被前置到讲义层，避免学生做成“材料堆积”。
3. `AI输出审核与人工复核记录` 被接入作业与讲义系统。
4. 在无正式班级反馈前，已预先接入课程级高频误区：
   - 关系链不足
   - 证据不清晰
   - 证据与结论不对应

## 5. 迁移结论

第八章 5.0 讲义已经满足：

`4.0 主线继承 + 5.0 profile 对齐 + 5.0 assignment pack 对齐`

但它当前仍属于“已迁移、待课堂回归”的状态。后续应在本章第一次真实项目批改后，继续用教学反馈和 memory 进行二次修订。
