## 4. 5.0 新增升级点

与 4.0 相比，第一章 5.0 讲义新增了这些系统约束：

1. `系统链路图/关系图` 被显式提升为收口证据，而不是项目附图。
2. `证据 -> 结论` 约束被前置到讲义层，避免学生只会背概念。
3. `AI输出审核与人工复核记录` 被接入作业与讲义系统。
4. 当前已预先接入课程级高频误区：
   - 任务部分完成
   - 证据不清晰
   - 证据与结论不对应
   - 关系链不足
   - 边界表达不足

## 5. 迁移结论

第一章 5.0 讲义已经满足：

`4.0 主线继承 + 5.0 profile 对齐 + 5.0 assignment pack 对齐`
