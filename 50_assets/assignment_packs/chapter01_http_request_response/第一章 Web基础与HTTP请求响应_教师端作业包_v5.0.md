# 第一章 Web基础与HTTP请求响应 教师端作业包 v5.0

## 一、基础信息

- 课程：Web应用安全与防护
- 章节：第一章 Web基础与HTTP请求响应
- 章节主线：从浏览器—服务器—数据库到请求、状态与统一分析框架
- profile 来源：`chapter01_http_request_response.json`

## 二、授权边界

1. 所有观察对象仅限 DVWA、本机浏览器、localhost、127.0.0.1 或教师明确授权环境。
2. 第一章不做任何公网对象验证、扫描或抓包演示。
3. 第一章先建立“技术动作必须服从授权边界”的课程基线。

## 三、能力目标

- 说出浏览器、服务器、数据库在 Web 应用中的基本角色
- 解释一次 HTTP 请求与响应的大致流程
- 识别 URL 参数、表单参数、Cookie、Session 等基础对象
- 解释系统为什么需要状态维持，以及 Cookie 与 Session 的分工
- 用“输入—处理—输出—执行—身份”框架分析简单页面场景

## 十二、关系图/关系表正反例提示

1. 错误写法：只列工具、术语、概念，或只列结果，不写关系。
2. 合格写法：至少写出“上一层结果如何导向下一步”。
3. 较好写法：写出“现象/输入 -> 处理/机制 -> 结果 -> 控制点 -> 下一步”。
4. 与 系统链路图/关系图 相关的结论必须体现“能说明什么 / 不能替代什么 / 下一步为什么是这个”。
