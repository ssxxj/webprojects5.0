# 第4到第8章 4.0讲义 -> 5.0作业包 对照表

## 1. 目的

本表不用于重复抄录 `projects4.0` 讲义，而用于确认：

1. `projects5.0` 生成的作业包是否沿用了 `projects4.0` 讲义中的章节主线。
2. `projects5.0` 的任务、关系图/关系表、评分与反馈链是否与 `projects4.0` 的章节目标和项目交付物一致。
3. 哪些内容是 `5.0` 在保持教学目标不变前提下新增的系统升级项。

## 2. 总体结论

总体判断：`第4到第8章的 5.0 作业包，与 4.0 讲义在章节定位、能力目标、章节项目方向上是对齐的。`

对齐方式不是“逐字复制讲义”，而是：

1. 保留 `4.0` 的章节主线与项目交付物方向。
2. 将其压缩为 `5.0` 的 Step11 标准输出：
   - 教师端作业包
   - 学生版作业说明单
   - 教师验收表
   - 班级级教学反馈模板
   - 任务分值与红线摘要
   - manifest
3. 在不改变教学目标的前提下，新增 `5.0` 的系统要求：
   - AI输出审核与人工复核记录
   - 红线与维度扣分分离
   - 关系图/关系表三档评分
   - 诊断标签
   - 班级级教学反馈接口

## 4. 第四章对照

### 4.1 4.0 来源

- 来源讲义：`projects4.0/50_assets/章节讲义/第四章 XSS漏洞与防护-按OMDM修订稿.md`
- 章节主线：
  - 浏览器不仅显示内容，还会解析和执行内容。
  - 同样的输入，在不同输出位置会有不同风险。
  - XSS 的防护重点不是猜输入，而是控制浏览器如何解释输出。
- 章节目标关键词：
  - Reflected / Stored / DOM XSS 差异
  - DVWA 基础观察
  - 浏览器解析、输出位置、脚本执行关系
  - 输出编码、输入校验、CSP
- 章节项目：
  - XSS 类型对照表
  - 类型/形成路径/风险/防护重点分析
  - XSS 防护清单

### 4.2 5.0 落点

- profile：`40_evaluation/runtime/chapter_profiles/chapter04_xss.json`
- 作业包目录：`50_assets/assignment_packs/chapter04_xss/`
- 任务拆分：
  1. 浏览器解析与输出上下文基础说明
  2. Reflected XSS Low 现象记录
  3. Stored XSS Low 现象记录
  4. DOM XSS 与类型差异分析
  5. Low/Medium 对照与 XSS 防护清单
- 收口项：
  - `XSS类型差异与风险链关系图/表`
  - `学生自评表`
  - `AI输出审核与人工复核记录`

### 4.3 对齐结论

对齐。  
4.0 的讲义重点是“浏览器解析 + 三类 XSS 路径 + Low/Medium 比较 + 防护清单”，5.0 作业包完整保留了这一教学主线，并将其拆成可评分的任务。

### 4.4 5.0 的新增升级

1. 把“类型差异—风险链条—防护建议”的结构，显式落成关系图/关系表评分项。
2. 增加 `AI输出审核与人工复核记录`，检查学生是否审核过 AI 输出。
3. 将“是否越权测试、是否未脱敏、是否雷同”从一般扣分问题升级为红线复核项。

## 5. 第五章对照

- 章节项目：

  - 风险点—控制点对照表
  - 上传防护清单

  4. 上传防护清单与业务方案设计

## 9. 仍需注意的边界

本次对照完成的是 `主线对齐、目标对齐、项目交付物方向对齐`。  
还没有做的是：

1. `4.0` 每个单元讲义正文到 `5.0` 单个任务字段的逐句映射。
2. `4.0` 教师提示语与 `5.0` 教师验收表的逐条对照。
3. `4.0` 章节讲义正文整体迁移到 `5.0` 章节讲义资产。

因此，这份对照表支持当前结论：

- `assignment pack` 已可视为 `4.0` 讲义目标在 `5.0 Step11` 中的正式落点。
- 但它还不能替代未来需要单独生成的 `5.0 综合章节讲义`。

## 10. 结论

当前应把 `5.0 assignment pack` 理解为：

`4.0 章节讲义的作业与评估执行面升级版`

而不是：

`4.0 章节讲义正文的完整替身`

这也是为什么下一步应优先做：

1. 把 `assignment_packs` 挂回 `50_assets/README_v5.0.md`
2. 补目录索引
3. 再升级 `skills` 与 `12步运行手册` 的引用关系

而不是直接把 `4.0` 全量讲义复制成 `5.0` 正文。
